# Activités Pratiques — Architecture JEE & Systèmes Distribués
### Professeur : Mohamed YOUSSFI — ENSET Mohammedia

---

Ce dépôt contient 2 travaux pratiques réalisés dans le cadre du cours d'Architecture JEE et Systèmes Distribués.

Chaque TP part de zéro et construit quelque chose de concret, étape par étape.

---

## N1 — Injection des Dépendances

On apprend comment faire communiquer les classes entre elles sans qu'elles se connaissent directement. C'est le principe de base de Spring.

---

## N2 — ORM JPA Hibernate Spring Data

On apprend à sauvegarder et récupérer des données depuis une base de données (H2 puis MySQL) sans écrire de SQL à la main.

---
## N3 — Application Web Spring Boot

On construit une vraie application web avec des pages, un formulaire, une connexion sécurisée et une base de données.

---
## N4 — Angular Framework

On construit une vraie application web avec des pages, un formulaire, une connexion sécurisée et une base de données.

---

## Technologies utilisées

- **Java 17+** et **Maven**
- **Spring Boot 3.2**
- **Spring Data JPA** + **Hibernate**
- **H2** (base en mémoire) et **MySQL**
- **Thymeleaf** + **Bootstrap 5**
- **Spring Security**
